# Gulp | Sass | Browser-Sync | Pug | Babel Boilerplate

## Clone the git repo
 ```
 git clone https://github.com/ibrodev/gulp-front-end.git
 ```

 ## Change directory to gulp-front-end

 ```
 cd gulp-front-end
 ```

 ## Install dependency packages

 ```
 npm install 
 ```
 or
 ```
 npm i
 ```
 ## Start the gulp default task

 ```
 gulp
 ```
